== Installation ==
1. From “add new plugin” Upload the Progress Bar WP Plugin zipped file to install it.
2. Activate the plugin using the 'Plugins' menu in your WordPress admin dashboard.
3. You can setup the progress bar WP from the “Appearances” “Widgets” menu.
4. You can generate a shortcode to insert a progress bar into a Post or Page by clicking on the “Progress Bar” setting in the menu.

== Frequently Asked Questions ==

= What is Progress Bar plugin? =

Progress bar plugin is helpful to create progress bar on any page or post. This plugin shows skills in percentage in animated styling.

= How it's work? =

You will need to just Add a new progess bar and set values and other attribute and than copy shortcode of progress bar. After publish it. Copy this code in any post or pages. 


= How to display it on Page? =

Just create progress bar and after publish it copy shortcode and just paste this code on any page or paste. After update the page. Progress bar will be display.
